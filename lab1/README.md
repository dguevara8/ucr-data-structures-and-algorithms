# Lab1_EDAI_template

# Laboratorio 1: Intro a C++, OOP, Clases y Objetos

Este repositorio contiene el código y la estructura para el Laboratorio 1, que se centra en conceptos básicos de programación y programación orientada a objetos.

## Estructura
- `src/`: Archivos fuente para cada ejercicio.
- `include/`: Archivos de cabecera para código reutilizable.
- `build/`: Aquí se almacenarán los archivos compilados.
- `docs/`: Documentación o instrucciones adicionales.
- `test/`: Pruebas unitarias para los ejercicios de ser necesarias

## Cómo Compilar
Cada ejercicio debería poder ser compilado a través del uso del Makefile
